# 🎙️ AI Meeting Summarizer

**High-Performance Transcription Analysis Engine**

A full-stack application that transforms chaotic meeting transcripts into structured, actionable insights using Llama 3.1 (Groq) and FastAPI.

## 🚀 Key Engineering Features

- **Model Agnostic Architecture:** Easily swap between Groq (Llama), OpenAI, or Anthropic.
- **Async Processing:** FastAPI backend handles long-form transcripts without blocking.
- **Type-Safe Frontend:** Built with Next.js 14 and strict TypeScript for zero-runtime errors.
- **Secrets Management:** Implements industry-standard environment variable masking.

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), TailwindCSS, TypeScript.
- **Backend:** FastAPI (Python), Pydantic (Validation).
- **AI:** Groq Cloud (Llama 3.1-8b).
